/usr/local/styles

This extension does NOT have a UI for editing styles; it's for people who want to use their text editor. If you would prefer to edit styles on this page, ChromeStylist is a great and easy-to-use extension. If you want to take the red pill, read on...

Where to find your userstyles

  1. Refer to this page to locate your user data directory.
  2. Within that directory, open the directory for this extension: Extensions/xxx
  3. Inside that one, open the directory for the latest version (currently 1.2)
  4. In that directory, you will see a directory named styles
  5. Create a symlink or shortcut to this directory from a more convenient location.

Editing

Three styles will be loaded for each page: __all__.css (put any styles you want applied to every site here), one for the domain, minus any leading "www." (example.com.css), and one for the specific page. For the latter, use subdirectories to match the path of the page you want to style (e.g. google.com/calendar/render.css).
